# 🫀 Heart Disease Risk Prediction

## 📌 Project Overview
Proyek ini merupakan end-to-end data science project yang bertujuan untuk menganalisis faktor-faktor yang memengaruhi risiko penyakit jantung serta membangun model machine learning untuk memprediksi kemungkinan pasien mengalami penyakit jantung.

Proyek ini membantu dalam **deteksi dini pasien berisiko tinggi** sehingga dapat mendukung pengambilan keputusan medis yang lebih cepat dan tepat.

---

## 🎯 Business Problem
Rumah sakit ingin:
- Mengidentifikasi faktor utama penyebab risiko penyakit jantung
- Mendeteksi pasien berisiko tinggi sejak dini
- Mengoptimalkan keputusan medis berbasis data

---

## 🎯 Project Objective
- Melakukan eksplorasi data pasien (EDA)
- Membersihkan dan mempersiapkan data
- Membangun model machine learning untuk prediksi risiko
- Menghasilkan insight yang dapat digunakan secara bisnis/medis

---

## 📊 Dataset
Dataset berisi data kesehatan pasien seperti:
- Age (usia)
- Sex (jenis kelamin)
- Chest Pain Type
- Blood Pressure
- Cholesterol
- Maximum Heart Rate
- Indikator medis lainnya
